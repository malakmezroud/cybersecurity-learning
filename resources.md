# Cybersecurity Learning Resources

Curated list of free and paid resources I'm using to prepare for my M.S. Cybersecurity at Humber College.

---

## 🎮 Hands-On Platforms

| Platform | Cost | Best For |
|---|---|---|
| [TryHackMe](https://tryhackme.com) | Free + Paid | Guided rooms, SOC Level 1, Pre-Security path |
| [HackTheBox](https://hackthebox.com) | Free + Paid | More advanced labs and CTFs |
| [CyberDefenders](https://cyberdefenders.org) | Free | Blue team / SOC analyst labs |
| [LetsDefend](https://letsdefend.io) | Free + Paid | SOC analyst simulation |
| [Cisco Skills for All](https://skillsforall.com) | Free | Networking, cybersecurity fundamentals |

## 🛠 Tools to Practice

- **Wireshark** — packet analysis
- **Nmap** — network scanning
- **Splunk** (free tier) — SIEM and log analysis
- **Kali Linux** (VirtualBox) — security testing OS
- **Burp Suite Community** — web app testing
- **VirtualBox** — home lab environment

---

## 🧠 Key Frameworks to Know

- [MITRE ATT&CK](https://attack.mitre.org) — attack taxonomy
- [NIST Cybersecurity Framework](https://www.nist.gov/cyberframework) — Identify, Protect, Detect, Respond, Recover
- [Cyber Kill Chain](https://www.lockheedmartin.com/en-us/capabilities/cyber/cyber-kill-chain.html) — 7 stages of an attack
- [OWASP Top 10](https://owasp.org) — web vulnerabilities
